# Weather Monitoring Application

A full-stack weather monitoring application built with Django REST Framework backend and React frontend.

## Live Demo
- **Frontend:** http://localhost:5173 (run locally)
- **Backend API:** http://127.0.0.1:8000/api/observations/
- **Admin Panel:** http://127.0.0.1:8000/admin/ (admin/admin123)

## Features
### CRUD Operations
- Create, Read, Update, Delete weather observations
- RESTful API endpoints

### API Integration
- OpenWeatherMap API integration
- Real-time weather data fetching

### Data Visualization
- Interactive charts with Chart.js
- Statistics dashboard

## Tech Stack
- **Backend:** Django, Django REST Framework, PostgreSQL
- **Frontend:** React, Vite, Chart.js, Bootstrap
